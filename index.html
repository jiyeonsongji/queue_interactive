항목내용
원리FIFO
삽입enQueue (rear)
삭제deQueue (front)
초기front=rear=-1
포화rear = n-1

🔄 원형 큐 (Circular)

항목내용
원리FIFO
삽입(rear+1) mod n
삭제(front+1) mod n
초기front=rear=0
포화(rear+1)%n=front

↔️ 데크 (Deque)

항목내용
원리양방향
삽입front 또는 rear
삭제front 또는 rear
Scroll입력 한쪽, 삭제 양쪽
Shelf입력 양쪽, 삭제 한쪽
자료구조 분류
분류자료구조
선형 구조스택, 큐, 데크, 배열, 연결 리스트
비선형 구조그래프, 트리
